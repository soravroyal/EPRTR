Packagecom.esri.solutions.flexviewer.utils
Classpublic class Hashtable

A hash table using direct lookup (perfect hashing). Each key can only map one value at a time and multiple keys can map the same value. The Hashtable is preallocated according to an initial size, but afterwards automatically resized if the number of key-value pairs exceeds the predefined size.

InheritanceEventBus flash.events.EventDispatcher

The EventBus allows centrallized communication among modules without point-to-point messaging. It uses the singleton design pattern to make sure one event bus is available globally. The bus itself is only available to the container. Modules use the container's static method to communicate with the event bus.
